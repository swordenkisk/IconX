IconX
v3.2.1 MIT 847 icons TypeScript zero deps

Static icons feel dead.
IconX icons feel alive.

Every icon breathes, pulses, orbits, glitches, and reacts to your cursor — because a good icon is not a picture. It's a presence. 847 icons. Zero dependencies. One line to make them live.

npm install iconx
🔥 Browse & click to copy hover to awaken · click to copy SVG
📦 Usage
// import any icon
import { HeartIcon, StarIcon, ZapIcon } from 'iconx'

// use it — it's already alive
<HeartIcon size={24} animation="heartbeat" />

// or go manual
<ZapIcon size={32} color="#f0883e" animation="pulse" orbit />
✨ Animation Modes
◆ heartbeat — lifelike pulse
◆ breathe — slow expand/contract
◆ float — gentle levitation
◆ spin — continuous rotation
◆ wiggle — playful shake
◆ orbit — particle ring
◆ glitch — cyberpunk flicker
◆ neon — neon flicker

◆ magnetic — cursor attraction
