**Design direction locked:** The concept is a GitHub-style icon library repo — but instead of dead, flat SVGs in a boring grid, every icon *breathes*, *pulses*, *reacts*, and *lives*. Dark terminal aesthetic meets organic motion. Icons glow, orbit particles, react to hover with magnetic pull. The repo looks like a real GitHub project page but the icons are alive creatures, not corpses.

**IconX** is alive. Here's what's breathing inside it:

---

### The repo looks like GitHub. The icons don't act like it.

Every icon has its own **personality and animation** — nothing is static:

| Animation | Icons |
|---|---|
| 💓 **Heartbeat** | Heart, Play, Dollar, Database |
| 🌬️ **Breathe** | Home, User, Message, Wifi, Code |
| 🌊 **Float** | Star, Music, Moon, Leaf, Box, Link |
| 🌀 **Spin** | Search, Settings, Refresh, CPU, Pie Chart |
| 🤪 **Wiggle** | Bell, Lock, Volume, Shuffle, Mail |
| 👁️ **Blink** | Eye, Camera, Terminal |
| ⚡ **Flicker** | Zap (neon flicker) |
| 🏀 **Bounce** | Arrow Up |
| ✨ **Orbit** | Zap, Play, Heart, Mic, Dollar, Share, Sun — with orbiting particles |

**Interactions:**
- **Hover** → magnetic pull follows your cursor
- **Click** → copies the SVG to clipboard + flash confirmation
- **Category tabs** → filter by Interface / Media / Arrows / Nature / Tech / Social / Finance
- **Search bar** → live filter by icon name
- **Star button** → click to star (turns gold)
- **Activity graph** → randomly generated commit heatmap
- **Scan-line overlay** + cursor glow follow you everywhere
